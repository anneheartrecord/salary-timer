> 打工人必备计时器，实时感受搬砖价值 🧱

## 功能

- 输入月薪、休息方式、每日工作时长，一键开始计时
- 实时显示今天赚的钱（精确到分钱），每秒跳动
- 显示每小时、每分钟、日薪等拆解数据
- 「摸鱼」按钮暂停计时
- 随机轮播搞怪提示，背景飘落钱符号

## 使用

直接用浏览器打开 `index.html`，或访问 https://anneheartrecord.github.io/salary-timer/。

## 本地运行

```bash
python3 -m http.server 3456
# 然后打开 http://localhost:3456
```
