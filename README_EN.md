> A real-time salary timer for the everyday worker 🧱

## Features

- Enter your monthly salary, rest days, and daily work hours
- Real-time display of today's earnings (down to fractions of a cent), updated every second
- Shows hourly rate, per-minute rate, and daily salary breakdown
- "Slack off" button to pause the timer
- Random funny tips that cycle every 5 seconds
- Animated falling money symbols in the background

## Usage

Open `index.html` directly in your browser, or visit the GitHub Pages deployment.

## Local Development

```bash
python3 -m http.server 3456
# Then open http://localhost:3456
```
